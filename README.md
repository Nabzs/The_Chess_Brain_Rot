# Jeu d'echecs 2D

Ce projet lance une version 2D du jeu d'echecs avec une interface ImGui.
Le projet applicatif ne conserve plus qu'un rendu en 2 dimensions.

## Build (Windows)

1. Configurer et compiler avec CMake.
2. L'executable est genere dans:

- `bin/Debug/ImGuiTemplate.exe`

## Lancement

Lancer l'executable:

- `bin/Debug/ImGuiTemplate.exe`

Le rendu du plateau et des pieces est gere par le renderer 2D.
